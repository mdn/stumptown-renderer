The -webkit-transform-2d Boolean CSS media feature is a Chrome extension whose value is true if vendor-prefixed CSS 2D {{cssxref("transform")}}s are supported.

Note: This media feature is only supported by WebKit. If possible, use an {{cssxref("@supports")}} feature query instead.

Specifications

Not part of any specification. Apple has a description in Safari CSS Reference.
