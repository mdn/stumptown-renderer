The -webkit-animation Boolean CSS media feature is a Chrome extension whose value is true if vendor-prefixed CSS {{cssxref("animation")}}s are supported.

Note: This media feature is only supported by WebKit. If possible, use an {{cssxref("@supports")}} feature query instead.

Example

@media (-webkit-animation) {
  /* CSS to use if animations are supported */
}

Specification

Not part of any specification. Apple has a description in Safari CSS Reference.
