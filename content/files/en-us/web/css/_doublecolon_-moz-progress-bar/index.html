The ::-moz-progress-bar CSS pseudo-element is a Mozilla extension that represents the progress bar inside a {{HTMLElement("progress")}}Â element. (The bar represents the amount of progress that has been made.)

Syntax

{{csssyntax}}

Examples

HTML

<progress value="30" max="100">30%</progress>
<progress max="100">Indeterminate</progress>

CSS

::-moz-progress-bar {
  background-color: red;
}

/* Force indeterminate bars to have zero width */
:indeterminate::-moz-progress-bar {
  width: 0;
}
