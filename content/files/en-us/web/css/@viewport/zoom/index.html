{{CSSRef}}

The zoom CSS descriptor sets the initial zoom factor of a document defined by the {{cssxref("@viewport")}} at-rule.

A zoom factor of 1.0 or 100% corresponds to no zooming. Larger values are zoomed in. Smaller values are zoomed out.

{{cssinfo}}

Syntax

/* Keyword value */
zoom: auto;

/* <number> values */
zoom: 0.8;
zoom: 2.0;

/* <percentage> values */
zoom: 150%;

Values

auto
The {{glossary("user agent")}} will set the document's initial zoom factor. The user agent may use the size of canvas area on which the document is rendered to determine that factor.
{{cssxref("<number>")}}
A non-negative number used as the zoom factor.
{{cssxref("<percentage>")}}
A non-negative percentage value used as the zoom factor.
