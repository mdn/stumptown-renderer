{{CSSRef}}

The min-zoom CSS descriptor sets the minimum zoom factor of a document defined by the {{cssxref("@viewport")}} at-rule. The browser will not zoom out any further than this, whether automatically or at the user's request.

A zoom factor of 1.0 or 100% corresponds to no zooming. Larger values are zoomed in. Smaller values are zoomed out.

{{cssinfo}}

Syntax

/* Keyword value */
min-zoom: auto;

/* <number> values */
min-zoom: 0.8;
min-zoom: 2.0;

/* <percentage> value */
min-zoom: 150%;

Values

auto
The {{glossary("user agent")}} will set the document's lower zoom factor limit.
{{cssxref("<number>")}}
A non-negative number limiting the minimum value of the zoom factor.
{{cssxref("<percentage>")}}
A non-negative percentage limiting the minimum value of the zoom factor.
