The ::-webkit-progress-inner-element CSS pseudo-element represents the outermost container of the {{HTMLElement("progress")}} element. It is the parent of the {{cssxref("::-webkit-progress-bar")}} pseudo-element.

Note: In order to let ::-webkit-progress-value take effect, {{cssxref("-webkit-appearance")}} needs to be set to none on the <progress> element.

Example

HTML

<progress value="10" max="50">

CSS

progress {
  -webkit-appearance: none;
}

::-webkit-progress-inner-element {
  border: 2px solid black;
}

Specifications

Not part of any specification. This is a proprietary pseudo-element specific to WebKit/Blink.
