The ::-moz-range-progress CSS pseudo-element is a Mozilla extension that represents the lower portion of the track (i.e., groove) in which the indicator slides in an {{HTMLElement("input")}} of type="range". This portion corresponds to values lower than the value currently selected by the thumb (i.e., virtual knob).

Note: Using ::-moz-range-progress with anything but an <input type="range"> doesn't match anything and has no effect.

Syntax

{{csssyntax}}

Example

HTML

<input type="range" min="0" max="100" step="5" value="50"/>
