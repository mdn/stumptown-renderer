The max-inline-size CSS property defines the horizontal or vertical maximum size of an element's block depending on its writing mode. It corresponds to the {{cssxref("max-width")}} or the {{cssxref("max-height")}} property depending on the value defined for {{cssxref("writing-mode")}}. If the writing mode is vertically oriented, the value of max-inline-size relates to the maximal height of the element, otherwise it relates to the maximal width of the element. It relates to {{cssxref("max-block-size")}}, which defines the other dimension of the element.

Syntax

/* <length> values */
max-inline-size: 300px;
max-inline-size: 25em;

/* <percentage> values */
max-inline-size: 75%;

/* Keyword values */
max-inline-size: auto;
max-inline-size: max-content;
max-inline-size: min-content;
max-inline-size: fit-content(20em);

/* Global values */
max-inline-size: inherit;
max-inline-size: initial;
max-inline-size: unset;

{{cssinfo}}

Values

The max-inline-size property takes the same values as the {{cssxref("max-width")}} and {{cssxref("max-height")}} properties.

Formal syntax

{{csssyntax}}

Example

HTML

<p class="exampleText">Example text</p>

CSS

.exampleText {
  writing-mode: vertical-rl;
  background-color: yellow;
  block-size: 100%;
  max-inline-size: 200px;
}
