The <translation-value> CSS data type is used in the arguments for certain {{cssxref("transform")}} functions, including translate(), translateX(), translateY(), and translate3d().

Syntax

The <translation-value> data type can be either a {{Cssxref("<length>")}} value or a {{Cssxref("<percentage>")}} value.
