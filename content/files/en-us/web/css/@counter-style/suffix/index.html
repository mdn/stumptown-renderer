The suffix descriptor of the{{cssxref("@counter-style")}} rule specifies content that will be appended to the marker representation. If not specified, the default value will be "\2E\20" (". ", a full stop followed by a space).

{{cssinfo}}

Syntax

/* <symbol> values */
suffix: "";
suffix: ") ";
suffix: url(bullet.png);

Values

<symbol>
Specifies a <symbol> that is appended to the marker representation. It may be a {{cssxref("<string>")}}, {{cssxref("<image>")}}, or {{cssxref("<custom-ident>")}}.

Formal syntax

{{csssyntax}}

Example

HTML

<ul class="choices">
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
  <li>None of the above</li>
</ul>

CSS

@counter-style options {
  system: fixed;
  symbols: A B C D;
  suffix: ") ";
}

.choices {
  list-style: options;
}
