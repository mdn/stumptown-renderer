The ::-webkit-meter-bar CSS pseudo-class is a WebKit extension that represents the meter bar in a {{HTMLElement("meter")}} element.

Specifications

Not part of any specification. This is a proprietary pseudo-element specific to WebKit/Blink.

Examples

HTML

<meter min="0" max="10" value="6">Score out of 10</meter>

CSS

meter {
  /* Reset the default appearance */
  -webkit-appearance: none;
  -moz-appearance: none;
  appearance: none;
}

meter::-webkit-meter-bar {
  background: #eee;
  box-shadow: 0 2px 3px rgba(0, 0, 0, 0.2) inset;
  border-radius: 3px;
}
