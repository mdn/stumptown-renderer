In Mozilla applications, the -moz-outline-radius-topleft CSS property can be used to round the top-left corner of an element's {{cssxref("outline")}}.

{{cssinfo}}

Example

HTML

<p>Look at this paragraph's top-left corner.</p>

CSS

p {
  margin: 5px;
  border: solid cyan;
  outline: dotted red;
  -moz-outline-radius-topleft: 2em;
}
