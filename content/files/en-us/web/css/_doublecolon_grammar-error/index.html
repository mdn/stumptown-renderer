The ::grammar-error CSS pseudo-element represents a text segment which the {{glossary("user agent")}} has flagged as grammatically incorrect.

::grammar-error {
  color: green;
}

Allowable properties

Only a small subset of CSS properties can be used in a rule with ::grammar-error in its selector:

  • {{cssxref("color")}}
  • {{cssxref("background-color")}}
  • {{cssxref("cursor")}}
  • {{cssxref("caret-color")}}
  • {{cssxref("outline")}} and its longhands
  • {{cssxref("text-decoration")}} and its associated properties
  • {{cssxref("text-emphasis-color")}}
  • {{cssxref("text-shadow")}}

Syntax

::grammar-error
