The ::-webkit-progress-value CSS pseudo-element represents the filled-in portion of the bar of a {{HTMLElement("progress")}} element. It is a child of the {{cssxref("::-webkit-progress-bar")}} pseudo-element.

Note: In order to let ::-webkit-progress-value take effect, {{cssxref("-webkit-appearance")}} needs to be set to none on the <progress> element.

Specifications

Not part of any specification. This is a proprietary pseudo-element specific to WebKit/Blink.

Example

HTML

<progress value="10" max="50">

CSS

progress {
  -webkit-appearance: none;
}

::-webkit-progress-value {
  background-color: orange;
}
