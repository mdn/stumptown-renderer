The CSS ID selector matches an element based on the value of its {{htmlattrxref("id")}} attribute. In order for the element to be selected, its id attribute must match exactly the value given in the selector.

/* The element with id="demo" */
#demo {
  border: red 2px solid;
}

Syntax

#id_value { style properties }

Note that this is equivalent to the following {{Cssxref("Attribute_selectors", "attribute selector")}}:

[id=id_value] { style properties }

Example

CSS

#identified {
  background-color: skyblue;
}

HTML

<div id="identified">This div has a special ID on it!</div>
<div>This is just a regular div.</div>
