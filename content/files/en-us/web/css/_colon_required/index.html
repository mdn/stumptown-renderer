The :required CSS pseudo-class represents any {{HTMLElement("input")}}, {{HTMLElement("select")}}, or {{HTMLElement("textarea")}} element that has the {{ htmlattrxref("required", "input") }} attribute set on it.

/* Selects any required <input> */
input:required {
  border: 1px dashed red;
}

This pseudo-class is useful for highlighting fields that must have valid data before a form can be submitted.

Note: The {{cssxref(":optional")}} pseudo-class selects optional form fields.

Accessibility concerns

Mandatory {{htmlelement("input")}}s should have the {{ htmlattrxref("required", "input") }} attribute applied to them. This will ensure that people navigating with the aid of assistive technology such as a screen reader will be able to understand which inputs need valid content to ensure a successful submission.

If the form also contains optional inputs, required inputs should be indicated visually using a treatment that does not rely solely on color to convey meaning. Typically, descriptive text and/or an icon are used.
