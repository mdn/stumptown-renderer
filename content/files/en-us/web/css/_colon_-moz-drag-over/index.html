The :-moz-drag-over CSS pseudo-class is a Mozilla extension that matches an element when a {{event("dragover")}} event is called on it.

Syntax

:-moz-drag-over

Example

HTML

<table border="1">
  <tr>
    <td width="100px" height="100px">Drag Over</td>
  </tr>
</table>

CSS

td:-moz-drag-over {
  color: red;
}
