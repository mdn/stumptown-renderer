The round function returns a number that is the nearest integer to the given number.

Syntax

round(decimal )

Arguments

decimal
The decimal number to be rounded.

Returns

The nearest integer less then, greater than, or equal todecimal.

Notes

  • -0.5 rounds to negative zero. 0.4 rounds to postive zero.

Defined

XPath 1.0 4.4

Gecko support

Supported.
