The Math.exp() function returns ex, where x is the argument, and e is {{jsxref("Math.E", "Euler's number (also known as Napier's constant)", "", 1)}}, the base of the natural logarithms.

Syntax

Math.exp(x)

Parameters

x
A number.

Return value

A number representing ex, where e is {{jsxref("Math.E", "Euler's number", "", 1)}} and x is the argument.

Description

Because exp() is a static method of Math, you always use it as Math.exp(), rather than as a method of a Math object you created (Math is not a constructor).

Examples

Using Math.exp()

Math.exp(-1); // 0.36787944117144233
Math.exp(0);  // 1
Math.exp(1);  // 2.718281828459045
