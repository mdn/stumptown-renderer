Syntax

bool.valueOf()

Return value

The primitive value of the given {{jsxref("Boolean")}} object

Description

The valueOf method of {{jsxref("Boolean")}} returns the primitive value of a {{jsxref("Boolean")}} object or literal {{jsxref("Boolean")}} as a Boolean data type.

This method is usually called internally by JavaScript and not explicitly in code.

Examples

Using valueOf

x = new Boolean();
myVar = x.valueOf(); // assigns false to myVar
