The HTTP {{HTTPHeader("Feature-Policy")}} header field's payment directive controls whether the current document is allowed to use the Payment Request API. When this policy is disabled, the {{DOMxRef("PaymentRequest()")}} constructor will throw a {{DOMxRef("SecurityError")}}.

Syntax

Feature-Policy: payment <allowlist>;
<allowlist>
{{page("Web/HTTP/Feature_Policy/Using_Feature_Policy", "allowlist")}}

Default policy

The payment feature's default allowlist value is 'self'.
