The HTTP {{HTTPHeader("Feature-Policy")}} header midi directive controls whether the current document is allowed to use the Web MIDI API. When this policy is enabled, the {{jsxref("Promise")}} returned by {{domxref("Navigator.requestMIDIAccess()")}} will reject with a {{domxref("DOMException")}}.

Syntax

Feature-Policy: midi <allowlist>;
<allowlist>
{{page("Web/HTTP/Feature_Policy/Using_Feature_Policy", "allowlist")}} The default value is 'self'.
