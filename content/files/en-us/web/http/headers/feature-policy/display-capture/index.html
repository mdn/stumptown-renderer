The HTTP {{HTTPHeader("Feature-Policy")}} headerÂ display-capture directive controls whether or not the document is permitted to use Screen Capture API, i.e.,{{domxref("MediaDevices.getDisplayMedia", "getDisplayMedia()")}} to capture the screen's contents.

If display-capture is disabled in a document, the document will not be able to initiate screen capture via {{domxref("MediaDevices.getDisplayMedia", "getDisplayMedia()")}}.

Syntax

Feature-Policy: display-capture <allowlist>;
