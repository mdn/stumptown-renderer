The {{Glossary("HTML")}} {{HTMLElement("img")}} element lets you embed images into an HTML document, the {{HTMLElement("picture")}} element enables responsive images. In this guide you will find links to resources that deal with adding images to websites.

References

HTML

These articles cover HTML features for using images on the web.

{{HTMLElement("img")}}
The <img> element is used to embed an image into a webpage. It supports a wide range of attributes to control how the image behaves and add important information such as {{htmlattrxref("alt", "img")}} text for people who do not see the image.
{{HTMLElement("picture")}}
The HTML <picture> element contains zero or more {{HTMLElement("source")}} elements and one {{HTMLElement("img")}} element to provide versions of an image for different display/device scenarios. The browser will consider each child <source> element and choose the best match among them.

CSS

{{cssxref("object-fit")}}
The object-fit CSS property sets how the content of a replaced element, such as an <img> or <video>, should be resized to fit its container.
{{cssxref("object-position")}}
The object-position CSS property specifies the alignment of the selected replaced element's contents within the element's box. Areas of the box which aren't covered by the replaced element's object will show the element's background.
{{cssxref("background-image")}}
The background-image CSS property sets one or more background images on an element.
