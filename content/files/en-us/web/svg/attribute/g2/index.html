The g2 attribute specifies a list of glyph names which identify a set of possible second glyphs in the kerning pair.

All glyphs with the given glyph name are included in the set. The total set of possible secondÂ  glyphs in the kerning pair is the union of glyphs specified by the {{SVGAttr("u2")}} and g2 attributes.

Two elements are using this attribute: {{SVGElement("hkern")}} and {{SVGElement("vkern")}}
