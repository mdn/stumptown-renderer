The pageLeft read-only property of the {{domxref("VisualViewport")}} interface returns the x coordinate of the left edge of the visual viewport relative to the initial containing block origin, in CSS pixels.

Syntax

var pageLeft = VisualViewport.pageLeft

Value

A double.
