The onerror property of the {{domxref("SpeechRecognition")}} interface represents an event handler that will run when a speech recognition error occurs (when the error event fires.)

Syntax

mySpeechRecognition.onerror = function() { ... };

Examples

var recognition = new SpeechRecognition();

recognition.onerror = function(event) {
  console.log('Speech recognition error detected: ' + event.error);
}
