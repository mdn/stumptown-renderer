The onerror property of the {{domxref("Notification")}} interface specifies an event listener to receive {{event("error")}} events. These events occur when something goes wrong with a {{domxref("Notification")}} (in many cases an error preventing the notification from being displayed.)

Syntax

Notification.onerror = function() { ... };

Value

A {{jsxref("function")}} which serves as the event handler for the {{event("error")}} event. When an error occurs, the specified function will be called. If null, no error handler is in effect.
