{{APIRef("Media Session API")}}

The Media Session API's MediaImage dictionary describes the images associated with the media resource {{domxref("MediaMetadata")}}. Its contents can be displayed by the {{Glossary("user agent")}} in appropriate contexts, like player interface to show the current playing video or audio track.

Properties

{{domxref("MediaImage.src", "src")}}
The URL from which user agent can fetch the image's data.
{{domxref("MediaImage.sizes", "sizes")}}
Specifies the resource in multiple sizes so that user agent does not have to scale a single image.
{{domxref("MediaImage.type", "type")}}
The {{Glossary("MIME type")}} hint for the user agant. Note that it is just a hint so that user agent may ignore images of types it does not support; user agent still may use mime type sniffing after downloading the image to determine its type.
