{{APIRef("HTML DOM")}}

The persisted read-only property indicates if a webpage is loading from a cache.

Syntax

window.addEventListener('pageshow', function(event) {
  if (event.persisted) {
    console.log('Page was loaded from cache.');
  }
});

Value

A {{jsxref("Boolean")}}.
