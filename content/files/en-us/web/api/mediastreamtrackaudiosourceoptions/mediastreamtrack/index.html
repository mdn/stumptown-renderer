{{APIRef("Web Audio API")}}

The {{domxref("MediaStreamTrackAudioSourceOptions")}} dictionary's mediaStreamTrack property must contain a reference to the {{domxref("MediaStreamTrack")}} from which the MediaStreamTrackAudioSourceNode being created using the {{domxref("MediaStreamTrackAudioSourceNode.MediaStreamTrackAudioSourceNode", "MediaStreamTrackAudioSourceNode()")}} constructor.

Syntax

mediaStreamTrackAudioSourceOptions = {
  mediaStreamTrack: audioSourceTrack;
}

mediaStreamTrackAudioSourceOptions.mediaStreamTrack = audioSourceTrack;

Value

A {{domxref("MediaStreamTrack")}} from which the audio output of the new MediaStreamTrackAudioSourceNode will be taken.
