{{securecontext_header}}{{APIRef("Storage")}}

The NavigatorStorage.storage read-only property returns the singleton {{domxref("StorageManager")}} object used to access the overall storage capabilities of the browser for the current site or app. The returned object lets you examine and configure persistence of data stores and learn approximately how much more space your browser has available for local storage use.

Syntax

var storageManager = navigator.storage;

Value

A {{domxref("StorageManager")}} object you can use to maintain persistence for stored data, as well as to determine roughly how much room there is for data to be stored.
