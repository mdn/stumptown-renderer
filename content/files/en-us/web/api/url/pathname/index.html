{{ApiRef("URL API")}}

The pathname property of the {{domxref("URL")}} interface is a {{domxref("USVString")}} containing an initial '/' followed by the path of the URL (or the empty string if there is no path).

{{AvailableInWorkers}}

Syntax

const path = url.pathname
url.pathname = newPath

Value

A {{domxref("USVString")}}.

Examples

const url = new URL('https://developer.mozilla.org/en-US/docs/Web/API/URL/pathname?q=value');
console.log(url.pathname); // Logs "/en-US/docs/Web/API/URL/pathname"
