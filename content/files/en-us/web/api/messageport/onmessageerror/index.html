{{APIRef("HTML DOM")}}

The onmessageerror event handler of the {{domxref("MessagePort")}} interface is an {{domxref("EventListener")}}, called whenever an {{domxref("MessageEvent")}} of type messageerror is fired on the port—that is, when it receives a message that cannot be {{glossary("Deserialization", "deserialized")}}.

{{AvailableInWorkers}}

port.onmessageerror = function() { ... };
