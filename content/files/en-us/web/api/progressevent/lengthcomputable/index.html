{{APIRef("DOM Events")}}

The ProgressEvent.lengthComputable read-only property is a {{domxref("Boolean")}} flag indicating if the resource concerned by the {{domxref("ProgressEvent")}} has a length that can be calculated. If not, the {{domxref("ProgressEvent.total")}} property has no significant value.

Syntax

flag = ProgressEvent.lengthComputable
