The onselectionchange property of the {{domxref("GlobalEventHandlers")}} mixin is an {{domxref("EventHandler")}} that processes {{event("selectionchange")}} events.

The selectionchange event fires when the text selected on a webpage changes.

Syntax

object.onselectionchange = functionRef;

Value

functionRef is a function name or a function expression. The function receives a {{domxref("FocusEvent")}} object as its sole argument.

Example

let selection;

document.onselectionchange = function() {
  console.log('New selection made');
  selection = document.getSelection();
};
