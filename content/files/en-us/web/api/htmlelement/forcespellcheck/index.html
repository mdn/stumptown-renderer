The forceSpellCheck() method of the {{domxref("HTMLElement")}} interface forces a spelling and grammar check on HTML elements, even if the user has not focused on the elements. This method overrides {{glossary("user agent")}} behavior. The specific user interface of the check, for example whether red underlining appears, is determined by the user agent.

Syntax

element.forceSpellCheck()
