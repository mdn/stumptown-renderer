The URLUtilsReadOnly.origin read-only property is a {{domxref("DOMString")}} containing the Unicode serialization of the origin of the represented URL, that is, for http and https, the scheme followed by '://', followed by the domain, followed by ':', followed by the port (the default port, 80 and 443 respectively, if explicitely specified). For URL using file: scheme, the value is browser dependant.

This version of origin is implemented by {{domxref("WorkerLocation")}} for use on workers.

Syntax

string = object.origin;

Examples

// On this page, returns the origin
var result = self.location.origin; // Returns:'https://developer.mozilla.org:443'
