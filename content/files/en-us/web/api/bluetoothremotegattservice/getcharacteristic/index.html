The BluetoothGATTService.getCharacteristic() method returns a {{jsxref("Promise")}} to an instance of {{domxref("BluetoothGATTCharacteristic")}} for a given universally unique identifier (UUID). 

Syntax

bluetoothGATTServiceInstance.getCharacteristic(characteristic).then(function(BluetoothGATTCharacteristic) { ... } )

Returns

A {{jsxref("Promise")}} to an instance of {{domxref("BluetoothGATTCharacteristic")}}

Parameters

characteristic
The UUID of a characteristic, for example '00002a37-0000-1000-8000-00805f9b34fb' for the Heart Rate Measurement characteristic.
