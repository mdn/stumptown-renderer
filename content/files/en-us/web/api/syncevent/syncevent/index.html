The SyncEvent() constructor creates a new {{domxref("SyncEvent")}} object.

Syntax

var mySyncEvent = new SyncEvent(type, init)

Parameters

type
The type of the Event.
init {{optional_inline}}
An options object containing any custom settings that you want to apply to the event object. Options are as follows:
  • tag: A developer-defined unique identifier for this SyncEvent.
  • lastChance: A {{jsxref("Boolean")}} indicating that the user agent will not make further synchronization attempts after the current attempt.
