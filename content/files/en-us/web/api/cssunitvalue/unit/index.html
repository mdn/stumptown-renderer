The CSSUnitValue.unit read-only property of the {{domxref("CSSUnitValue")}} interface returns a {{jsxref('USVString')}} indicating the type of unit.

Syntax

var aString = CSSUnitValue.unit;

Value

A {{jsxref('USVString')}}.

Examples

The following creates a {{domxref('CSSPositionValue')}} from individual CSSUnitValue constructors, then queries the CSSUnitValue.unit.

let pos = new CSSPositionValue(
    new CSSUnitValue( 5, "px" ),
    new CSSUnitValue( 10, "em" ));

console.log( pos.x.unit ); // "px"
console.log( pos.y.unit ); // "em"
