{{APIRef("Service Workers API")}}

The navigate() method of the {{domxref("WindowClient")}} interface loads a specified URL into a controlled client page then returns a {{jsxref("Promise")}} that resolves to the existing {{domxref("WindowClient")}}.

Syntax

windowClient.navigate(url).then(function(windowClient) {
  // do something with your WindowClient after navigation
});

Parameters

url
The location to navigate to.

Return value

A {{jsxref("Promise")}} that resolves to the existing {{domxref("WindowClient")}}.
