The RTCAnswerOptions dictionary is used to provide optional settings when creating an {{Glossary("SDP")}} answer usingÂ {{domxref("RTCPeerConnection.createOffer()")}} after receiving an offer from a remote peer. The createOffer() method's options parameter is of this type.

Properties

This dictionary inherits properties from the {{domxref("RTCOfferAnswerOptions")}} dictionary, on which it's based.

Note: At this time, RTCAnswerOptions does not have any additional properties defined beyond those included in {{domxref("RTCOfferAnswerOptions")}}. However, this is likely to change in the future, so the type has been defined in preparation for that eventuality.
