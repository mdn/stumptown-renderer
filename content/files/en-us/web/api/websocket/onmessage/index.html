{{APIRef("Web Sockets API")}}

The WebSocket.onmessage property is an {{domxref("EventHandler")}} that is called when a message is received from the server. It is called with a {{domxref("MessageEvent")}}.

Syntax

aWebSocket.onmessage = function(event) {
  console.debug("WebSocket message received:", event);
};

Value

An {{domxref("EventListener")}}.
