The store() method of the {{domxref("CredentialsContainer")}} stores a set of credentials for the user inside a {{domxref("Credential")}} instance, returning this in a {{domxref("Promise")}}.

This method is restricted to top-level contexts. Calls to it within an <iframe> element will resolve without effect.

Syntax

CredentialsContainer.store(Credential).then(function(Credential) { ... } )

Parameters

Credentials
A valid {{domxref("Credential")}} instance.

Returns

A {{domxref("Promise")}}, resolving to the passed {{domxref("Credential")}} instance.
