{{APIRef("HTML DOM")}}

The source read-only property of the {{domxref("MessageEvent")}} interface is a MessageEventSource (which can be a {{domxref("WindowProxy")}}, {{domxref("MessagePort")}}, or {{domxref("ServiceWorker")}} object) representing the message emitter.

Syntax

let mySource = messageEvent.source;

Value

a MessageEventSource (which can be a {{domxref("WindowProxy")}}, {{domxref("MessagePort")}}, or {{domxref("ServiceWorker")}} object) representing the message emitter.

Example

myWorker.onmessage = function(e) {
  result.textContent = e.data;
  console.log('Message received from worker');
  console.log(e.source);
};
