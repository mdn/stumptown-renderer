{{APIRef("HTML DOM")}}

The origin read-only property of the {{domxref("MessageEvent")}} interface is a {{domxref("USVString")}} representing the origin of the message emitter.

Syntax

var origin = messageEvent.origin;

Value

A {{domxref("USVString")}} representing the origin.

Example

myWorker.onmessage = function(e) {
  result.textContent = e.data;
  console.log('Message received from worker');
  console.log(e.origin);
};
