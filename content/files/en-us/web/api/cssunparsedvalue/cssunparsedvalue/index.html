The CSSUnparsedValue() constructor creates a new {{domxref("CSSUnparsedValue")}} object which represents property values that reference custom properties.Â 

Syntax

var CSSUnparsedValue = new CSSUnparsedValue(members)

Parameters

members
An array whose values must be either a {{domxref('USVString')}} or a {{domxref('CSSVariableReferenceValue')}}.

Examples

let value  = new CSSUnparsedValue( ['4deg'] ),
    values = new CSSUnparsedValue( ['1em', '#445566', '-45px'] );

console.log( value  ); // CSSUnparsedValue {0: "4deg", length: 1}
console.log( values ); // CSSUnparsedValue {0: "1em", 1: "#445566", 2: "-45px", length: 3}
