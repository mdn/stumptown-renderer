The hasOrientation read-only property of the {{domxref("VRPositionState")}} interface returns a boolean indicating whether the {{domxref("VRPositionState.orientation")}} property is valid (i.e. if the hardware is currently registering a valid orientation). If it is false, the orientation property will return null.

Syntax

var myHasOrientation = VRPositionState.hasOrientation;

Value

A {{domxref("Boolean")}}.
