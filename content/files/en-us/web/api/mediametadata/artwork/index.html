The artwork property of the {{domxref("MediaMetaData")}} interface returns or sets an array of {{domxref("MediaImage")}} objects representing images associated with playing media.

Syntax

var artwork[] = mediaMetadata.artwork
mediaMetadata.artwork = artwork[]

Value

An {{jsxref("Array")}} of {{domxref("MediaImage")}} objects.
