The MediaKeySession.load() method returns a {{jsxref('Promise')}} that resolves to a boolean value after loading data for a specified session object.

Syntax

mediaKeySession.load(sessionId).then(function(booleanValue) { ... });

Parameter

sessionId
A unique string generated by the content decription module for the current media object and its associated keys or licenses.

Return value

A {{jsxref('Promise')}} that resolves to a boolean indicating whether the load succeeded or failed.
