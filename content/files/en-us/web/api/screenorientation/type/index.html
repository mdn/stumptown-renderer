The type read-only property of the {{domxref("ScreenOrientation")}} interface returns the document's current orientation type, one of "portrait-primary", "portrait-secondary", "landscape-primary", or "landscape-secondary".

Syntax

type = ScreenOrientation.type

Value

A {{jsxref("String")}}.
