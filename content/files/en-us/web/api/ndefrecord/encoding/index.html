The encoding property ofthe {{DOMxRef("NDEFRecord")}} interface is {{DOMxRef("USVString")}} containing the name of the encoding used to encode NDEF payload if it contains textual data.

Syntax

NDEFRecord.encoding

Value

A {{DOMxRef("USVString")}} which can be one of the following: "utf-8", "utf-16", "utf-16le" or "utf-16be".
