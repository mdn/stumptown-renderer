The scrollBy() method of the {{domxref("Element")}} interface scrolls an element by the given amount.

Syntax

element.scrollBy(x-coord, y-coord);
element.scrollBy(options)

Parameters

  • x-coord is the horizontal pixel value that you want to scroll by.
  • y-coord is the vertical pixel value that you want to scroll by.

- or -

  • options is a {{domxref("ScrollToOptions")}} dictionary.

Examples

// scroll an element
element.scrollBy(300, 300);

Using options:

element.scrollBy({
  top: 100,
  left: 100,
  behavior: 'smooth'
});
