The SVGScriptElement interface corresponds to the SVG {{SVGElement("script")}} element.

{{InheritanceDiagram(600, 140)}}

Properties

{{domxref("SVGScriptElement.type")}} {{ReadOnlyInline}}
A {{domxref("DOMString")}} corresponding to the {{SVGAttr("type")}} attribute of the given {{SVGElement("script")}} element. A {{domxref("DOMException")}} is raised with the code NO_MODIFICATION_ALLOWED_ERR on an attempt to change the value of a read only attribut.
{{domxref("SVGScriptElement.crossOrigin")}} {{ReadOnlyInline}}
A {{domxref("DOMString")}} corresponding to the {{SVGAttr("crossorigin")}} attribute of the given {{SVGElement("script")}} element.

Methods

This interface doesn't implement any specific methods, but inherits methods from its parent interface, {{domxref("SVGElement")}}.
