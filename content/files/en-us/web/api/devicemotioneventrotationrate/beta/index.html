This property indicates the rate of rotation around the X axis -- in degrees per second -- in a {{ domxref("DeviceMotionEventRotationRate") }} object.

Syntax

var beta = deviceRotationRate.beta;

This property is read-only.

Return value

beta
A double indicating the rate of rotation around the X axis, in degrees per second. See Accelerometer values explained for details.
