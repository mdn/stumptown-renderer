The rssi read-only property of the {{DOMxRef('BluetoothAdvertisingData')}} interface returns the power at which the device’s packets are being received, measured in dBm. This is used to compute the path loss as this.txPower - this.rssi.

Syntax

var rssi = BluetoothAdvertisingData.rssi;
