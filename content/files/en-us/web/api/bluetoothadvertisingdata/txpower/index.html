The txPower read-only property of the {{DOMxRef('BluetoothAdvertisingData')}} interface returns the transmission power at which the device is broadcasting, measured in dBm. This is used to compute the path loss as this.txPower - this.rssi.

Syntax

var power = BluetoothAdvertisingData.txPower;
