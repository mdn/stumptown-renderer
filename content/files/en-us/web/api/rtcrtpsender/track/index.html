{{APIRef("WebRTC")}}

The track read-only property of the {{domxref("RTCRtpSender")}} interface returns the {{domxref("MediaStreamTrack")}} which is being handled by the RTCRtpSender. If track is null, the RTCRtpSender doesn't transmit anything.

Syntax

var mediaStreamTrack = RTCRtpSender.track

Value

A {{domxref("MediaStreamTrack")}} object.
