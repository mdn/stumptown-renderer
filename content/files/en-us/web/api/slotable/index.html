{{APIRef("Shadow DOM")}}

The Slotable mixin defines features that allow nodes to become the contents of a {{htmlelement("slot")}} element — the following features are included in both {{domxref("Element")}} and {{domxref("Text")}}.

Properties

{{domxref("Slotable.assignedSlot")}} {{readonlyInline}}
Returns the {{htmlelement("slot")}} the node is inserted in.

Methods

None.
