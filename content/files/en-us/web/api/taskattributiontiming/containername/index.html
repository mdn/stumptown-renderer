The containerName readonly property of the {{domxref("TaskAttributionTiming")}} interface returns the container's name attribute. A container is the iframe, embed or object etc. that is being implicated, on the whole, for a long task.

Syntax

var containerName = TaskAttributionTiming.containerName;

Value

A {{domxref("DOMString")}} containing the container's name attribute.
