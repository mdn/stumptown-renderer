The {{domxref("XRSessionEventInit")}} dictionary's session property specifies the {{domxref("XRSession")}} for which the event describes a state change.

Syntax

let sessionEventInit = {
  session: xrSession
};
mySessionEvent = new XRSessionEvent(type, sessionEventInit);

mySessionEvent = new XRSessionEvent(type,
                 { session: XRSession });

Value

An {{domxref("XRSession")}} object indicating which WebXR session the event is referring to.
