The value property of {{domxref("DeviceProximityEvent")}} objects provides the current distance between the device and the detected object, in centimeters.

Syntax

var distance = instanceOfDeviceProximityEvent.value;

Value

A positive number representing a distance in centimeters (cm) between the device's proximity sensor and the detected object.
