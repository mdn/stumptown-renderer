{{APIRef("WebAPI")}}

The {{domxref("RTCTrackEventInit")}} dictionary's track property specifies the {{domxref("MediaStreamTrack")}} associated with the {{event("track")}} event.

Syntax

var trackEventInit = {
  receiver: rtpReceiver,
  track: mediaStreamTrack,
  streams: [videoStream],
  transceiver: rtpTransceiver
};

var track = trackEventInit.track;

Value

A {{domxref("MediaStreamTrack")}} representing the track with which the event is associated. This is the track that's being added to the {{domxref("RTCPeerConnection")}}.
