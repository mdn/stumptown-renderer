The iconURL read-only property of the {{domxref("PasswordCredential")}} interface returns a {{domxref("USVString")}} containing a URL pointing to an image for an icon. This image is intended for display in a credential chooser. The URL must be accessible without authentication.

Syntax

url =passwordCredential.iconURL

Value

A {{domxref("USVString")}} containing a URL.
