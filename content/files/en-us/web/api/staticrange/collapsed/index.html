{{APIRef("DOM WHATWG")}}

The collapsed read-only property of the {{domxref("StaticRange")}} interface returns true if the range's start position and end position are the same.

Syntax

var isCollpased = staticRange.collapsed

Value

A {{jsxref("Boolean")}} value which is true if the range is collapsed. A collapsed range is one in which the start and end positions are the same, resulting in a zero-character-long range..
