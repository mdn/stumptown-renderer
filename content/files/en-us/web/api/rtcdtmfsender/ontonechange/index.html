The ontonechange property of the {{ domxref("RTCDTMFSender") }} interface is used to set the event handler for the {{event("tonechange")}} event, which is sent to the RTCDTMFSender each time a tone begins or ends. The event handler receives as input a single parameter of type {{domxref("RTCDTMFToneChangeEvent")}}, which describes the change.

Syntax

RTCDTMFSender.ontonechange = toneChangeHandlerFunction;

Value

A function which is called when a {{event("tonechange")}} event is sent to the RTCDTMFSender, indicating that a DTMF tone has either started playing, or if all tones have finished playing.
