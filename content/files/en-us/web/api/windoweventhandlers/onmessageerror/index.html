{{APIRef("HTML DOM")}}

The onmessageerror event handler of the {{domxref("WindowEventHandlers")}} interface is an {{domxref("EventListener")}}, called whenever an {{domxref("MessageEvent")}} of type messageerror is fired on a window—that is, when it receives a message that cannot be {{glossary("Deserialization", "deserialized")}}.

{{AvailableInWorkers}}

Syntax

window.onmessageerror = function() { ... };
