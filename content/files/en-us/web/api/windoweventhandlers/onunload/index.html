{{APIRef("HTML DOM")}}

The onunload property of the {{domxref("WindowEventHandlers")}} mixin is the {{domxref("EventHandler")}} for processing {{Event("unload")}} events. These events fire when the window is unloading its content and resources. The resource removal is processed after the unload event occurs.

Note: Browsers equipped with pop-up blockers will ignore all {{domxref("Window.open()")}} method calls in onunload event handler functions.

Syntax

window.addEventListener("unload", function(event) { ... });
window.onunload = function(event) { ... };

Typically, it is better to use {{domxref("EventTarget.addEventListener", "window.addEventListener()")}} and the {{event("unload")}} event, instead of onunload.
