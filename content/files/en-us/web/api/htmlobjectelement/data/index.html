The data property of the {{domxref("HTMLObjectElement")}} interface returns a {{domxref("DOMString")}} that reflects the {{htmlattrxref("data", "object")}} HTML attribute, specifying the address of a resource's data.

Syntax

var data = HTMLObjectElement.data;
HTMLObjectElement.data;

Value

A {{domxref('DOMString')}}.
