The to() method of the {{domxref("CSSNumericValue")}} interface converts a numberic value from one unit to another.

Syntax

var cssUnitValue = CSSNumericValue.to(unit);

Parameters

unit
The unit to which you want to convert.

Return value

A {{domxref('CSSMathSum')}}.

Exceptions

SyntaxError
Indicates that an invalid type was passed to the method.
TypeError
Indicates that the passed values cannot be summed.

Examples

// Prints "0.608542cm"
console.log(CSS.px("23").to("com").toString());
