The Window.sizeToContent() method sizes the window according to its content. In order for it to work, the DOM content should be loaded when this function is called—for example, once the {{event("DOMContentLoaded")}} event has been thrown.

The minimal size of the window can be clamped.

Syntax

window.sizeToContent()

Example

window.sizeToContent();

Specification

This feature is not part of any specification.
