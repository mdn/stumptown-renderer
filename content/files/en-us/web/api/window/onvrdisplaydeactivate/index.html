The onvrdisplaydeactivate property of the {{domxref("Window")}} interface represents an event handler that will run when a display can no longer be presented to (when the {{event("vrdisplaydeactivate")}} event fires), for example if an HMD has gone into standby or sleep mode due to a period of inactivity.

The event object is of type {{domxref("VRDisplayEvent")}}.

Syntax

window.onvrdisplaydeactivate = function() { ... };

Examples

window.onvrdisplaydeactivate = function() {
  info.textContent = 'Display deactivated.';
};
