The Window.onuserproxymity property represents an {{domxref("EventHandler")}}, that is a function to be called when the {{event("userproximity")}} event occurs. These events are of type {{domxref("UserProximityEvent")}} and occur when the the device sensor detects that an object becomes nearby.

Syntax

window.onuserproximity = eventHandler
