{{APIRef("Service Workers API")}}

The onupdatefound property of the {{domxref("ServiceWorkerRegistration")}} interface is an {{domxref("EventListener")}} property called whenever an event of type statechange is fired; it is fired any time the {{domxref("ServiceWorkerRegistration.installing")}} property acquires a new service worker.

Note: This feature is available in Web Workers.

Syntax

serviceWorkerRegistration.onupdatefound = function() { ... };
