{{APIRef("Web Workers API")}}

The onerror property of the {{domxref("WorkerGlobalScope")}} interface represents an {{domxref("EventHandler")}} to be called when the {{event("error")}} event occurs and bubbles through the {{domxref("Worker")}}.

Syntax

self.onerror = function() { ... };

Example

The following code snippet shows an onerror handler set inside a worker:

self.onerror = function() {
  console.log('There is an error inside your worker!');
}
