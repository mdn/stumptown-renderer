The HTMLInputElement.mozSetFileNameArray() method sets the names of the files that selected on an HTML input element.

Note: This method is Gecko-specific and is not available in other browsers. Also, it throws an error if used in Web pages.

Syntax

inputElement.mozSetFileNameArray(aFileNames, aLength);

Parameters

aFileNames
Is the array of file names to apply to the element.
aLength
Is the number of file names in the array.

Example

var fileArray = {"/foo/bar.txt", "/foo/foosball.txt"};

inputElement.mozSetFileNameArray(fileArray, fileArray.length);

Specification

Not part of any specification; Mozilla-specific method.
